# **App Name**: Arjun's Echoes

## Core Features:

- Tech Page: Dedicated page to showcase Arjun's coding projects, skills, and experience in technology.
- Singing Portfolio: Showcase Arjun's singing abilities with audio samples, performance videos, and repertoire list.
- Piano Performance: Dedicated page to show Arjun's Piano playing with performance audio/video and song lists.
- Swimming Achievements: Page highlighting Arjun's swimming journey, achievements, and training experiences, complete with images and videos.
- Badminton Highlights: Page featuring Arjun's badminton journey, accomplishments, and match highlights, complete with images and videos.
- Contact Form: A contact form and information on how to reach Arjun.
- AI-Powered Bio Generation: Use an AI tool to generate different versions of Arjun's biography based on the targeted audience or purpose (e.g., tech resume, music bio, sports introduction).
- Home Page: Landing page with an introduction and links to other sections.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey trust, depth, and sophistication.
- Background color: Light Grey (#F0F0F0) for a clean and modern feel.
- Accent color: Soft Teal (#64B5F6) to complement the primary color and add a touch of innovation and creativity.
- Headline font: 'Belleza' (sans-serif) for titles and headings, providing an elegant and stylish appearance.
- Body font: 'Alegreya' (serif) for body text, for readability and elegance.
- Use a set of modern, outline-style icons to represent different sections and skills.
- Employ a clean and structured layout with a focus on readability and easy navigation.
- The logo 'EOA' microphone is the main logo of the website.
